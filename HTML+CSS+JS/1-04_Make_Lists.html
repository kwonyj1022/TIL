목록을 생성할 때 사용할 수 있는 태그 : ul(unordered list), ol(ordered list), dl(description list) 태그

ul 태그 : 순서가 없는 비순서형 목록을 생성할 때 사용

  • 목록 내용은 li(list item) 태그로 구성
  • ul 태그를 사용하면 목록 내용마다 글머리 기호가 붙음

ol 태그 : 순서형 목록을 생성할 때 사용

  1. ul 태그와 마찬가지로 li 태그로 목록 내용 구성
  2. ol 태그를 사용하면 목록 내용에 번호가 붙음

dl 태그 : 정의형 목록을 만들 때 사용

정의형 목록
용어와 용어 설명을 나열한 형태의 목록으로, dt 태그와 dd 태그를 사용
dt 태그
description term의 약자로, 용어의 제목에 해당하는 내용을 작성할 때 쓰는 태그
dd 태그
description details의 약자로, 용어의 설명에 해당하는 내용을 작성할 때 쓰는 태그
