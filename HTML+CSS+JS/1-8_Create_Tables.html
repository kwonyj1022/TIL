표 : 2차원 격자 형태로 구성된 데이터 (행(row), 열(column), 셀(cell)로 구성됨)

table 태그 : 표 생성에 사용 (표 관련 태그는 모두 table 태그 안에서 사용) (자세한 설명)

참고) 표에서 요약 정보를 나타내는 summary 속성은 더 이상 사용하지 않음

caption 태그 : 표 제목 지정 (필수. table 태그 안에 첫 번째로 작성해야 함) (표를 생성할 때 웹 접근성을 향상하는 방법의 하나) (자세한 설명)

tr(table row) 태그 : 표에서 행 생성 (행을 여러 개 만들고 싶다면 tr 태그를 여러 번 사용하면 됨) (자세한 설명)

th(table header), td(table data) 태그 : 표에서 열 생성 (자세한 설명)

  • th(table header) 태그 : 표에서 제목을 나타내는 열을 생성할 때 사용
  • td(table data) 태그 : 표에서 일반적인 데이터를 나타내는 열을 생성할 때 사용
태그로 만든 표 예시
번호 상품명 수량 가격
1 콜라 1개 1,500원
2 사이다 2개 1,000원
3 탄산수 3개 1,000원

참고) HTML에서 표는 기본으로 테두리가 없게 설정되어 있기 때문에, 테두리가 있게 하려면 CSS를 적용해야 함

  • rowspan 속성 : 속성값으로 병합하고 싶은 셀의 개수를 적고, 병합할 셀의 개수만큼 다음 행은 비워야 함
  • colspan 속성 : 속성값으로 병합하고 싶은 셀의 개수를 적고, 병합할 셀의 개수만큼 다음 열은 비워야 함
태그로 만든 표 예시 (rowspan & colspan 포함)
번호 상품명 수량 가격
1 콜라 1개 1,500원
2 사이다 2개 1,000원
3 탄산수 3개
총 금액 6,500원
  • scope 속성 : 제목을 나타내는 셀의 범위 지정 (th 태그에서만 사용 가능) (순전히 웹 접근성 향상을 목적으로 사용)
    • 속성값 col : 제목의 범위는 세로 방향
    • 속성값 row : 제목의 범위는 가로 방향
    • 제목이 colspan이나 rowspan으로 병합된 셀이라면 col이나 row 대신 colgroup, rowgroup 사용
제목의 인식 범위를 설정한 표 예시
구분 중간고사 기말고사
전공 A+ B+
교양 C- B

thead, tfoot, tbody 태그 : 표에서 행을 그룹화할 때 사용 (자세한 설명)

  • thead 태그 : 헤더 영역에 해당하는 행 그룹화
  • tfoot 태그 : 푸터 영역에 해당하는 행 그룹화
  • tbody 태그 : 본문 영역에 해당하는 행 그룹화
  • thead, tfoot, tbody 태그는 사용하지 않을 수 있지만, 사용한다면 반드시 thead, tfoot, tbody 순서여야 함
  • thead와 tfoot 태그는 한 번만 사용할 수 있고, thead 태그로 그룹화한 행은 th 태그로 열을 생성해야 함
  • 사용하는 이유 1 : 웹 접근성 향상 (접근하는 표가 어떤 데이터를 담고 있는지 표 내용을 모두 읽기 전에 파악 가능)
  • 사용하는 이유 2 : 표를 인쇄할 때, 표 데이터가 여러 장에 걸쳐 있더라도 thead, tfoot 태그는 각각의 페이지에 인쇄돼어 여러 장에 걸쳐 있는 표가 어떤 표인지 쉽게 파악 가능
    • 태그로 만든 표 예시 (thead, tfoot, tbody 사용)
    번호 상품명 수량 가격

    1 콜라 1개 1,500원
    2 사이다 2개 1,000원
    3 탄산수 3개

col과 colgroup 태그 : 표에서 열을 그룹화할 때 사용 (자세한 설명)

  • col 태그 : 하나의 열 그룹화
  • colgroup 태그 : span 속성과 함께 사용해 2개 이상의 열 그룹화
  • caption 태그를 사용했다면 col 태그나 colgroup 태그는 caption 태그 다음에, tr 태그보다 먼저 사용해야 함
  • col 태그나 colgroup 태그를 사용하면 반드시 두 태그의 개수와 사용한 열의 개수가 일치해야 함
    • 태그로 만든 표 예시 (1열 너비 80px, 2,3열 너비 150px, 4열 너비 100px)
