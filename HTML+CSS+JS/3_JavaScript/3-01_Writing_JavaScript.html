자바스크립트 코드 작성 방법

HTML 파일과 자바스크립트 연결하기 자세한 설명

내부 스크립트 방법 : HTML 파일에서 script 태그의 콘텐츠 영역에 자바스크립트 코드를 작성하는 방법 자세한 설명

<body>
    <script>
        document.write("내부 스크립트 방법으로 작성한 것입니다")
    </script>
</body>

[실행결과]

외부 스크립트 방법 : js 확장자로 된 별도의 파일을 생성하고, 생성한 파일에 자바스크립트 코드를 작성한 뒤, HTML 문서와 연결하는 방법 자세한 설명

[.js파일]
document.write("외부 스크립트 방법으로 작성한 것입니다");
[HTML 파일]
<body>
    <script src="script.js">></script>
</body>

[실행결과]

script 태그 사용 위치 : body 태그의 종료 태그 바로 전에 사용하는 것이 가장 적절 자세한 설명

이유 : 웹 브라우저는 HTML 문서를 첫 번째 줄부터 순차적으로 해석하기 때문에,
        script 태그를 위쪽에 작성하면 자바스크립트 파일을 해석하는 데 시간이 오래 걸릴 경우
        웹 브라우저에 전체 내용이 표시되는 시간도 길어짐

자바스크립트 코드 실행하기 자세한 설명

콘솔창 사용하기 자세한 설명

  1. 크롬 브라우저에서 주소창에 about:blank를 입력해 빈 페이지 열기
  2. f12 눌러서 개발자 도구 열고 콘솔창(Console)으로 이동
  3. 커서가 깜박이는 곳에 자바스크립트 코드 입력 후 Enter키 누르면 자바스크립트 코드가 실행됨
  • undefined : 자바스크립트 코드 실행 결과로 반환해야 하는 값이 있으면 그 값을 보여주고, 없으면 undefined가 나옴

확장 프로그램 사용하기 자세한 설명

  1. Code Runner(ex. VSCode) 설치 후 자바스크립트 파일을 열어 실행 버튼 클릭
  2. 에디터 하단에 작성한 코드의 실행결과가 출력됨

프로그래밍 시작 전 알아둘 사항

주석 자세한 설명

  • 한 줄만 주석으로 처리할 때는 // 기호(슬래시 2개) 사용
    // 한 줄 주석은 슬래시 2개로 표시합니다
  • 여러 줄을 주석으로 처리하고 싶을 때는 /* 기호와 */ 기호 사이에 내용 작성
    /*
        자바스크립트는 여러 줄 주석도
        간단하게 처리할 수 있습니다
    */

자바스크립트 오류 확인 방법 자세한 설명

  • 자바스크립트는 인터프리터 방식으로 실행되기 때문에 오류 발생 시점부터 실행을 멈춤
  • 웹 브라우저 개발자 도구에서 콘솔창을 보면 오류 내용을 보여줌 ,
    (오류 메시지, 오류 발생 파일명, 코드의 줄 번호 포함)
  • 자바스크립트에서 발생할 수 있는 올 메시지와 해결 방법을 알고 싶다면 여기 참고
