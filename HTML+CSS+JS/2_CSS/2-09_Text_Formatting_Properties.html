텍스트 속성으로 텍스트 꾸미기

font-family 속성 : 글꼴 지정 자세한 설명

  • 속성값으로는 글꼴명을 적으면 되는데 쉼표(,)로 구분하고 1개 이상의 글꼴을 나열해야 함
  • 한글로 된 글꼴이나 공백이 있는 글꼴명은 큰따옴표("")로 항상 감싸야 함
  • 글꼴을 쉼표로 구분해 나열하면 맨 앞에 작성한 글꼴부터 적용 가능 여부를 판단
  • 예시 font-family:"Times New Roman", BatangChe; (Times New Roman 적용됨)
  • 반드시 글꼴 유형을 작성하도록 권장 (글꼴 유형은 글꼴의 개념이 아니라 글꼴의 형태를 의미하기 때문에 못 불러온다는 개념이 없음)
    (font-family 속성으로 명시한 글꼴을 불러오지 못할 경우를 대비해 불러오려고 한 글꼴과 가장 유사한 형태의 글꼴 유형을 마지막에 항상 지정해야 함
    -> 지정한 글꼴을 불러오지 못하더라도 텍스트가 유사한 형태로 보여짐)
  • 예시 font-family:dkdk, serif; (dkdk 글꼴이 없으므로 serif 글꼴 유형이 적용됨)
글꼴 유형
글꼴 유형 설명 예시
serif 삐침이 있는 명조 계열의 글꼴 안녕하세요 // Hello, everyone
sans-serif 삐침이 없고 굵기가 일정한 고딕 계열의 글꼴 안녕하세요 // Hello, everyone
monospace 텍스트 폭과 간격이 일정한 글꼴 안녕하세요 // Hello, everyone
fantasy 화려한 글꼴 안녕하세요 // Hello, everyone
cursive 손으로 쓴 것 같은 필기체 계열의 글꼴 안녕하세요 // Hello, everyone

font-size 속성 : 텍스트 크기 변경 자세한 설명

  • 속성값으로는 단위를 포함한 크기를 넣음
  • 예시 font-size:20px;

font-weight 속성 : 텍스트 굵기 지정 자세한 설명

  • 숫자 표기법 : 숫자 100 단위로 텍스트 굵기 표현 (100부터 900까지의 값 사용 (100이 가장 얇고 900이 가장 굵음))
    • font-weight:100; /* 최소 굵기 */
    • font-weight:900; /* 최대 굵기 */
  • 키워드 표기법 : 키워드로 텍스트 굵기 표현 (lighter, normal, bold, bolder)
    • font-weight:lighter; /* 부모 요소의 굵기보다 얇게 지정됨 (상대적인 값) */
    • font-weight:normal; /* 숫자 표기법에서 400과 같은 굵기
    • font-weight:bold; /* 숫자 표기법에서 700과 같은 굵기
    • font-weight:bolder; /* 부모 요소의 굵기보다 굵게 지정됨 (상대적인 값)

font-style 속성 : 글꼴 스타일 지정 자세한 설명

font-style 속성값
속성값 설명 예시
normal 기본 형태로 표시 안녕하세요 // Hello, everyone
italic 이탤릭체로 표시 안녕하세요 // Hello, everyone
oblique 기울임꼴로 표시 안녕하세요 // Hello, everyone
  • italic과 oblique은 차이가 별로 없음 (italic은 글꼴에서 지원하는 이탤릭체를 적용한 값이고, oblique은 단순히 글자를 기울여서 보여주는 것)

a 태그에 자동으로 생기는 밑줄을 제거하고 싶으면 tet-decoration 속성을 none 값으로 지정하면 됨

  • a 태그 밑줄 제거 안했을 때
  • a 태그 밑줄 제거 했을 때

text-decoration 속성값
속성값 설명 예시

letter-spacing 속성 : 자간 조절 자세한 설명

  • 속성값으로 normal 또는 크기를 넣을 수 있음
    • 속성값으로 normal을 넣으면 이렇게 됩니다
    • 속성값으로 크기를 넣으면 이렇게 됩니다. (지금 자간은 15px)

line-height 속성 : 행간(텍스트 한 줄의 높이) 지정 자세한 설명

line-height 속성값
속성값 설명
normal 웹 브라우저에서 정한 기본값 적용
숫자 현재 font-size 값에 입력한 숫자를 곱한 값 적용
퍼센트 현재 font-size 값에 입력한 비율을 곱한 값 적용
크기 입력한 크기 적용
  • line-height:normal; /* 웹 브라우저에서 정한 기본값, 보통 1.2 */
  • line-height:1; /* 현재 font-size 값의 1배
  • line-height:1.5; /* 현재 font-size 값의 1.5배
  • line-height:200%; /* 현재 font-size 값의 200% (2배)
  • line-height:16px;
  • line-height가 font-size보다 작으면
    윗줄과 아랫줄의 텍스트가 겹쳐보이면서 가독성이 떨어질 수 있으므로
    line-height는 속성값을 항상 font-size보다 크게 지정하는 편이 좋음
