배경 속성으로 요소의 배경 설정하기

박스 모델에서 padding과 content 영역은 요소의 배경을 나타내는 영역이고, 이러한 박스 모델의 배경에는 색상이나 이미지와 같은 스타일을 지정할 수 있음

background-color 속성 : 배경 색상 지정 (속성값으로는 색상값 사용) 자세한 설명

  • 요소의 배경 크기가 반드시 있어야 함

background-image 속성 : 배경에 이미지 삽입 (속성값은 이미지의 경로를 url()함수로 지정) 자세한 설명

  • 요소의 배경 크기가 반드시 있어야 함
  • 이미지의 크기가 요소의 배경 크기와 맞지 않으면 어색하게 보임
    • (배경 크기 < 이미지) => 원본 이미지가 일부 잘림
    • (배경 크기 > 이미지) => 이미지가 반복되어 채워짐

background-repeat 속성 : 배경 크기가 삽입하려는 이미지보다 크면 이미지를 자동으로 반복해서 채우는데, 이때의 이미지 반복 설정을 바꿀 때 사용 자세한 설명

background-repeat 속성값
속성값 설명
no-repeat 이미지를 반복하지 않음
repeat-x 이미지를 가로 방향으로 반복
repeat-y 이미지를 세로 방향으로 반복
repeat 이미지를 가로와 세로 방향으로 반복
round 이미지를 반복하되, 이미지가 요소에 딱 맞도록 크기를 자동 조절
space 이미지가 잘리지 않도록 반복

* 지금 배경은 round 속성 적용함 *

background-size 속성 : 배경 크기가 이미지보다 작을 때 이미지가 잘려 보이는 문제를 해결하기 위해, 이미지 크기를 지정 자세한 설명

background-size 속성값
속성값 설명
auto 이미지 크기 유지
cover 이미지의 가로 세로 비율을 유지하면서 크기를 확대하거나 축소해 요소의 배경에 꽉 채움
contain 이미지의 가로 세로 비율을 유지하면서 이미지가 배경 요소 안에 들어가도록 크기를 확대하거나 축소
너비 높이 이미지 크기 직접 지정
  • cover와 contain은 비슷해 보이지만, contain은 배경의 가로와 세로 중 한 부분이 가득 차면 그 상태에서 멈추고, 못 채운 부분은 이미지를 반복해서 채움
  • 너비와 높이를 직접 지정하는 방법 예시 -> background-size:80px 60px;

* 지금 배경은 contain 속성 적용함 *

background-position 속성 : 삽입하려는 이미지의 위치 결정 자세한 설명

  • 형식 background-position:x위치 y위치
  • 속성값은 1개만 사용해도 되고 2개를 사용해도 됨
    • 값을 1개만 사용하면 지정한 값은 x축 값이 되고, y축은 기본으로 cener가 됨
    • 값을 2개 사용하면 각각 x축과 y축 값으로 지정됨
background-position 속성값
위치 속성값 설명
x left, center, right x축(가로) 방향의 위치 지정
y top, center, bottom y축(세로) 방향의 위치 지정
공통 px, rem, em, % 위치를 직접 지정

background-attachment 속성 : 요소에 삽입된 이미지를 스크롤할 때, 이미지의 작동 방식 결정 자세한 설명

background-attachment 속성값
속성값 설명
local 삽입된 이미지가 요소와 웹 브라우저에서 모두 스크롤됨
scroll 삽입된 이미지가 요소에서는 고정되지만, 웹 브라우저에서는 스크롤됨
fixed 삽입된 이미지가 요소와 웹 브라우저에서 모두 고정됨

background 속성으로 한 번에 지정하기 자세한 설명

속성값을 나열하는 순서는 정해져 있지 않지만,
background-size 속성은 항상 background-position 속성값을 먼저 작성하고 나서 슬래시(/)로 구분한 뒤 사용해야 함

  • div{
    background-color:red;
    background-image:url('images/coffee.jpg');
    background-repeat:no-repeat;
    background-position:center;
    background-size:100% 100%;
    background-attachment:fixed;
    }
  • div{
    background:red url('./coffee.jpg') no-repeat center/1000% 100% fixed; }
  • 위 두 코드는 동일한 코드임
  • 나열 순서가 상관 없기 때문에 속성값의 의미를 한눈에 파악하기 어려우므로 되도록 background 속성을 사용하지 말고 속성을 하나씩 작성해서 사용하기를 권장
