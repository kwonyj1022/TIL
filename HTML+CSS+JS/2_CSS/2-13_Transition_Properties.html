전환 효과 속성 사용하기

전환 : 한 요소에 적용된 스타일 속성값을 다른 속성값으로 변하게 하는 것

transition-property 속성 : 전환 효과를 적용할 대상 속성 지정 자세한 설명

  • 속성값에는 전환 효과의 대상이 되는 속성명이나 미리 정해진 키워드를 넣음
  • 전환 효과를 여러 속성에 지정하고 싶다면 쉼표로 구분해 작성

transition-property 속성값
속성값 설명
none 전환 효과 속성을 지정하지 않음
all 모든 속성을 전환 효과 대상으로 지정
  • 사용 예1) transition-property:background-color; /* background-color 속성에 전환 효과를 줌 */
  • 사용 예2) transition-property:background-color, color, width; /* 전환 효과를 여러 속성에 지정

transition-duration 속성 : 전환 효과의 지속 시간 설정 (속성값으로는 초 단위 값 사용 (ex. 5s)) 자세한 설명

  • 여러 속성을 쉼표로 구분해 전환 효과를 지정할 때는 transition-property에 적힌 순서에 따라 적어야 함
    • transiton-property:background-color, color, width;
      transition-duration:1s, 500ms, 2s;
      이렇게 작성하면 background-color 속성은 1초, color 속성은 0.5초, width 속성은 2초 동안 지속 시간이 지정되어 전환효과 발생
  • 어떤 요소에 전환 효과를 지정하려면 반드시 transition-property 속성과 transition-duration 속성을 함께 사용해야 함

transition-delay 속성 : 전환 효과의 발생 지연 (속성값으로는 지연하고 싶은 시간을 적으면 됨) 자세한 설명

마우스를 올리면
2초 뒤 서서히
크기가 커지면서
파란색으로 변하고
마우스를 치우면
원래대로 돌아옵니다

transition-timing-function 속성 : 전환 효과의 진행 속도 지정 자세한 설명

  • 속성값으로는 정해진 키워드와 cubic-bezier() 함수 사용
cubic-bezier(p1, p2, p3, p4)사용자가 정의한 속도로 진행
transition-timing-function 속성값
속성값 설명
linear 처음 속도와 마지막 속도가 일정
ease 처음에는 속도가 점점 빨라지다가 중간부터 점점 느려짐
ease-in 처음에는 속도가 느리지만 완료될 때까지 점점 빨라짐
ease-out 처음에는 속도가 빠르지만 완료될 때까지 점점 느려짐
ease-in-out 처음에는 속도가 느리지만 점점 빨라지다가 다시 점점 느려짐
지정 안함
linear
ease
ease-in
ease-out
ease-in-out
  • 완료 시점은 똑같은데 완료까지 전환 효과의 속도가 다른 것임
  • cubic-bezier() 함수로 전환 속도를 직접 정할 수 있지만, 복잡하므로 크롬의 개발자 도구나 별도의 외부 사이트에서 계산하는 것이 편함

크롬 개발자 도구에서 속도 조절하기 자세한 설명

  1. 크롬의 개발자 도구에서 transition-timing-function 속성이 적용된 요소 선택
  2. 'transition-timing-function:' 옆 보라색 아이콘 클릭
  3. 직접 곡선 조정
  4. 곡선 아래의 cubic-bezier() 값을 복사해서 속성값으로 넣기

외부 사이트 이용하기 (https://cubic-bezier.com/)

transitioin 속성으로 한 번에 지정하기 자세한 설명

  • 형식 transition:property, duration, timing-function, delay
  • 아래 두 코드는 같은 것임
    • transition-property:width;
      transition-duration:1s;
      transition-timing-function:ease-in;
      transition-delay:1s;
    • transition:width, 1s, ease-in, 1s;
