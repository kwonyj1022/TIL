조합 선택자 사용하기

조합 선택자 : 기본 선택자와 함께 사용했을 때 선택자의 의미를 더 풍부하게 해 주는 선택자 방법

그룹 선택자 : 여러 선택자를 하나로 그룹 지을 때 사용 자세한 설명

  • p{color:red;}, #title{color:red}, .red{color:red;} 를 묶어서 p, #title, .red{color:red;}로 나타냄

자식 선택자 : 부모 요소의 하위에 있는 자식 요소에 스타일을 적용할 때 사용 자세한 설명

  • 2개 이상의 선택자가 사용되며, 선택자와 선택자는 > 기호로 구분
  • 형식 : 부모 선택자 > 자식 선택자{/* CSS 코드 */} HTML의 부모, 자식, 형제 관계를 모른다면?

하위 선택자 : 선택자의 범위를 특정 부모 요소의 하위 요소(자손)로 한정하는 방법 자세한 설명

  • 2개 이상의 선택자를 사용하고, 선택자와 선택자는 공백으로 구분
  • 형식 : 선택자1 선택자2 선택자3 ...{/* CSS 코드 */}

인접 형제 선택자 : 앞에서 지정한 선택자 요소 바로 다음에 있는 형제 관계 요소를 선택자로 지정 자세한 설명

참고) 인접 형제 선택자는 이전 요소 바로 다음에 등장하는 형제 요소를 선택
따라서, 이전 요소보다 먼저 등장한 요소는 선택 대상이 아니며, 인접하지 않은 형제 요소도 선택 대상이 아님

  • 2개 이상의 선택자를 사용하고, 선택자와 선택자는 + 기호로 구분
  • 형식 : 이전 선택자 + 대상 선택자{/* CSS 코드 */}

일반 형제 선택자 : 이전 선택자 뒤에 오는 형제 관계 요소를 모두 선택자로 지정 자세한 설명

  • 2개 이상의 선택자를 사용하며, 선택자와 선택자는 ~ 기호로 구분
  • 형식 : 이전 선택자 ~ 대상 선택자{/* CSS 코드 */}

참고) 일반 형제 선택자도 인접 형제 선택자와 마찬가지로, 이전 요소 다음에 등장하는 형제 관계으 요소를 대상으로 지정 (이전 요소보다 먼저 등장하는 형제 관계 요소는 선택자 지정 대상이 아님)
