위치 속성으로 HTML 요소 배치하기

position 속성 : HTML 요소를 기본 흐름에서 벗어나 좌푯값에 따라 배치할 때 사용 자세한 설명

position 속성값
속성값 설명
static 요소를 기본 흐름에 따라 배치 (좌표 속성 사용 불가, 위치 속성을 사용하지 않은 것과 동일)
relative 요소를 기본 흐름에 따라 배치하지만, 좌표 속성을 사용할 수 있음
(기준점은 요소의 왼쪽 위 모서리 (absolute는 기준점이 웹 브라우저의 왼쪽 위 모서리))
  • 좌표 속성 top : 위쪽을 기준으로 좌푯값 지정
  • 좌표 속성 right : 오른쪽을 기준으로 좌푯값 지정
  • 좌표 속성 bottom : 아래쪽을 기준으로 좌푯값 지정
  • 좌표 속성 left : 왼쪽을 기준으로 좌푯값 지정
absolute 요소를 기본 흐름에서 벗어나, 절대적인 좌표 위치에 따라 배치
(기준점은 웹 브라우저의 왼쪽 위 모서리 (relative는 기준점이 요소의 왼쪽 위 모서리))
fixed 요소를 기본 흐름에서 벗어나, 절대적인 좌표 위치에 따라 배치
단, 스크롤해도 해당 위치에 고정되어 있음
sticky 요소를 static 값처럼 기본 흐름에 따라 배치하지만,
지정한 좌표의 임계점에 이르면 fixed 값처럼 화면에 고정됨

초록색 칸 요소의 position 속성값을 relative로 하고, 좌표 속성 left는 50px를 적용해
요소가 왼쪽을 기준으로 100px 이동됨

초록색 칸 요소의 position 속성값을 relative로 하고, 좌표 속성 left는 100px를 적용, top은 50px를 적용해
요소가 왼쪽을 기준으로 100px 이동된 후 위쪽을 기준으로 100px 이동됨

초록색 칸 요소의 position 속성값을 absolute로 하고, 좌표 속성 left는 100px를 적용해
요소가 왼쪽을 기준으로 50px 이동됨 (absolute일 때는 top, bottom 속성을 사용하지 않으면 원래 있던 위치에서 가로로만 움직임)
- relative 값일 때는 요소가 이동하더라도 요소가 원래 위치에 있는 것처럼 유지되지만,
- absolute 값일 때는 요소를 움직이면 원래 있던 공간은 빈 공간으로 인식되어 파란색 요소가 초록색 요소가 원래 있던 위치로 올라옴

초록색 칸 요소의 position 속성값을 absolute로 하고, 좌표 속성 left는 100px를 적용, top은 100px를 적용해
요소가 왼쪽을 기준으로 100px 이동된 후 위쪽을 기준으로 100px 이동됨

position 속성값을 fixed로 적용하면 absolute와 똑같이 작동하는 것처럼 보이지만, 웹 브라우저를 스크롤하더라도 항상 같은 위치에 있음

fixed

position 속성값을 sticky로 적용하면, 스크롤하는 중에 일정 지점(임계점)이 되면 요소가 fixed 값처럼 작동하는

sticky

z-index 속성 : position 속성으로 배치한 요소의 z축 위치 결정 (속성값으로는 하나의 정숫값 사용)


위의 경우, 요소와 요소가 겹침 (초록색 요소가 빨간색 요소보다 위에 표시됨)
이때 요소들의 앞뒤 배치를 바꾸고 싶다면 z축 값을 지정하는 z-index 속성 사용
(속성값의 숫자가 클수록 요소가 더 앞쪽에 배치됨 (참고) 모든 요소는 기본으로 z-index 속성값이 0임)

요소를 원하는 위치에 배치하고 싶을 때 position 속성은 아주 효과적이지만, 화면의 모든 구성 요소를 position속성으로 배치하는 것은 지양해야 함
모든 요소는 최대한 기본 흐름을 유지하면서 정말 필요한 대상에만 position 속성을 사용해야 함

float 속성 : 요소를 화면에 배치하는 또 다른 방법 자세한 설명

float 속성값
속성값 설명
none float 속성을 적용하지 않음
left 대상 요소를 공중에 띄워 왼쪽에 배치하면서 다음에 오는 요소를 주변에 자연스럽게 배치
right 대상 요소를 공중에 띄워 오른쪽에 배치하면서 다음에 오는 요소를 주변에 자연스럽게 배치

float 속성 사용 시 문제점

float 속성이 대상 요소의 다음 요소에 영향을 주기 때문에 영향을 주지 않도록 float 속성을 해제해야 함
해제하는 방법 : clear 속성 사용 또는 overflow 속성 사용

clear 속성 : float 속성을 해제할 때 사용 자세한 설명

clear 속성값
속성값 설명
left float 속성의 left 값 해제
right float 속성의 right 값 해제
both float 속성의 left와 right 값 모두 해제

자식 요소가 부모 요소에 감싸져 있는 형태라면 ::after 가상 요소 선택자로 clear 속성을 적용할 수도 있음

위의 예시의 경우, 부모 요소가 자식 요소인 빨간색과 파란색 요소의 공간을 제대로 인식하지 못하고 있음
(float 속성을 적용하면 해당 자식 요소를 부모 요소가 제대로 인식하지 못함)
::after 가상 요소 선택자로 clear 속성을 적용하여 해결 가능

.parent::after {
content:""; /* 빈 콘텐츠 만들기
display:block; /* 박스 모델 성격을 블록 성격으로 변경 */
clear:left; /* clear 속성으로 float 속성의 left 값 해제
}
