박스 모델 : 모든 HTML 요소가 사각형의 박스로 둘러 쌓여 있다는 개념

박스 모델의 구성 요소
구성 요소 설명
margin 요소의 외부 여백 의미
border 요소의 테두리(경계선) 의미
padding 요소의 내부 여백 의미
content 요소의 내용 의미

박스 모델을 적절하게 제어하면 HTML 요소를 원하는 위치에 배치할 수 있음

margin 영역 : 박스모델에서 가장 외부에 있는 영역으로, 요소의 외부 여백 담당 자세한 설명

  • 속성으로는 margin-top, margin-right, margin-bottom, margin-left가 있고, 모든 방향을 통합해 margin 속성으로 사용할 수도 있음
  • 속성값에는 크기(ex. 20px)를 넣으면 됨
  • margin-top:20px;
  • margin-right:20px;
  • margin-bottom:20px;
  • margin-left:20px;
  • margin:20px 25px 30px 35px; // margin-top:20px; margin-right:25px margin-bottom:30px; margin-left:35px; 와 같음
  • margin:20px 25px 30px; // margin-top:20px; margin-right:25px; margin-bottom:30px; margin-left:25px; 와 같음 (margin-left는 margin-right와 같은 속성값을 가짐)
  • margin:20px 30px; // margin-top:20px; margin-right:30px; margin-bottom:20px; margin-left:30px 와 같음 (앞에 있는 속성 값은 margin-top과 margin-bottom을, 뒤에 있는 속성값은 margin-right와 margin-left를 의미
  • margin:30px; // margin-top:30px; margin-right:30px margin-bottom:30px; margin-left:30px; 와 같음 (모든 방향이 같은 속성값을 가짐)

margin 겹침 현상 : 인접한 margin 값이 둘 중 더 큰 값으로 통일되는 것

  • 어떤 두 요소가 있는데, 첫 번째 요소에는 margin-bottom이 20px가 적용됐고, 두 번째 요소에는 margin-top 값이 30px 적용됐을 때,
    두 요소 사이의 간격은 20px와 30px를 더한 50px가 되는 것이 아니라, 20px와 30px 중 큰 값인 30px가 적용됨

border 영역 : margin보다 안쪽에 있으며, 요소의 테두리(경계선) 담당 자세한 설명

  • margin 영역처럼 위쪽, 아래쪽, 오른쪽, 왼쪽 방향에 해당하는 속성이 있음 (border-top, border-right, border-bottom, border-left 속성)
  • 모든 방향을 한 번에 똑같은 값으로 적용하고 싶다면 border 속성을 사용하면 됨
  • border 속성은 margin 속성과 달리 여러 속성값을 복합적으로 사용 (속성 값으로 다른 속성의 값을 사용하는 속성을 단축 속성 또는 축약 속성이라고 함)
    • border 속성과 border-방향 속성 모두 border-width, border-style, color 속성값 사용
      • 형식 border:border-width border-style color;

border-style 속성 : 테두리 모양 지정

border-style 속성값
속성값 설명

padding 영역 : margin, border 영역보다 안쪽에 있으며 요소의 내부 여백 담당 자세한 설명

  • padding 영역에서 사용할 수 있는 속성은 margin 영역에서 사용하는 속성과 형식이 비슷함
  • padding-top:크기;
  • padding-right:크기;
  • padding-bottom:크기;
  • padding-left:크기;
  • padding:[padding-top] [padding-right] [padding-bottom] [padding-left];
  • padding:[padding-top] [padding-right & padding-left] [padding-bottom];
  • padding:[padding-top & padding-bottom] [padding-right & padding-left];
  • padding:[padding-top & padding-right & padding-bottom & padding-left];

content 영역 : 시작 태그와 종료 태그 사이에 사용된 콘텐츠가 속하는 영역 자세한 설명

  • content 영역 자체를 다루는 속성은 없고, 텍스트 속성으로 content 영역을 제어할 수 있음
  • width 속성 : content 영역의 너비 지정
  • height 속성 : content 영역의 높이 지정
  • width 속성으로 HTML 요소의 너비를 줄였을 때, margin-left:auto; margin-right:auto 로 margin 속성을 적용하면 수평방향으로 중앙에 HTML 요소 위치
  • 웹 브라우저에 표시된 요소의 너비와 높이는 border, padding, content 영역을 전부 포함한 크기가 되기 때문에 box-sizing 속성을 사용하면 편함
    box-sizing 속성값
    속성값 설명
    content-box width, height 속성의 적용 범위를 content 영역으로 제한
    border-box width, height 속성의 적용 범위를 border 영역으로 제한
    • box-sizing 속성을 border-box 값으로 지정하면 width, height 속성에 지정한 크기에 요소를 맞추기 위해 내부 content 영역의 너비와 높이가 자동으로 조절됨

박스 모델의 성격과 display 속성 자세한 설명

  • 블록 성격 : hn, p, div 태그를 사용했을 때 요소의 너비가 콘텐츠 유무와 상관 없이 항상 가로 한 줄을 다 차지
    (hn, p, div 태그를 여러 번 사용하면 무조건 줄 바꿈 됨)
  • 인라인 성격 : a, span, strong 태그를 사용했을 때 요소의 너비 콘텐츠 크기만큼만 차지하는 성격
    (a, span, strong 태그를 여러 번 사용할 때 웹 브라우저의 수평 공간이 남아 있으면 한 줄로 배치됨)
  • 인라인 블록 성격 : 인라인 성격처럼 요소의 너비가 콘텐츠의 크기만큼만 차지하지만, 그 외의 성격은 블록 성격을 가지는 복합적인 성격
    (img 태그를 사용할 때 웹 브라우저의 수평 공간이 남아있으면 한 줄로 배치됨)
블록 vs 인라인 vs 인라인 블록
블록 성격 인라인 성격 인라인 블록 성격
width 속성 O X O
height 속성 O X O
margin 속성 O D (좌우만 가능) O
padding 속성 O D (좌우만 가능) O

display 속성 : HTML 태그가 기본으로 가지고 있는 박스 모델의 성격을 변경 자세한 설명

  • 속성값으로는 block, inline, inline-block 사용
    ex) 블록 성격인 태그를 인라인 성격으로 바꾸고 싶다면 display 속성값을 inline으로 지정
    ex) 인라인 성격인 태그를 인라인 블록 성격으로 바꾸고 싶다면 display 속성값을 inline-block으로 지정
