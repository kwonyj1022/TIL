링크에는 a 태그를, 이미지 객체 삽입에는 img 태그 사용

a 태그 : HTML에서 내부나 외부 링크 생성

* a 태그에 대한 자세한 설명을 보려면 여기를 클릭하세요

  • href 속성
    : a 태그로 생성하는 링크의 대상 경로를 입력할 때 사용 (생략 불가)
    • 필수로 사용해야 함 (target, title 속성은 선택적 사용)
    • 속성값은 대상 경로의 주소(URL)이거나 내부 문서의 id 속성값일 수 있음
    • 대상 경로가 명확하지 않을 때는 속성값에 "#"을 넣고, 나중에 올바른 대상경로로 변경해야 함
    * href 속성에 대한 자세한 설명을 보려면 여기를 클릭하세요
  • target 속성
    : a 태그로 링크를 생성할 때 대상이 연결되는 방식 지정 (생략 가능)
    • 속성 값으로 _blank, _self, _parent, _top 사용 가능
      • _blank 속성 : 새 윈도우 창을 열어 웹페이지를 연다 (기존 창은 그대로 남아 있다) 예시
      • _self 속성 : 현재 윈도우 창에 웹페이지를 연다 예시
      • _parent 속성 : 현재 프레임의 부모 프레임에서 새 웹페이지를 연다 예시
      • _top 속성 : 최상위 프레임에서 새 웹페이지를 연다 예시
    • _blank 속성을 제외하고 거의 사용하지 않음
    * target 속성에 대한 자세한 설명을 보려면 여기를 클릭하세요
  • title 속성
    : 링크를 설명할 수 있는 텍스트 작성 (생략 가능)
    * title 속성에 대한 자세한 설명을 보려면 여기를 클릭하세요

img 태그 : 이미지를 삽입할 때 사용

  • src 속성(필수)
    : 삽입하려는 이미지의 경로를 입력하는 속성 (웹 브라우저에서 실행되는 HTML 파일의 위치가 기준)
  • alt 속성(필수)
    : 삽입한 이미지 객체를 설명할 수 있는 텍스트 입력 (그림 위에 마우스를 올려두면 확인할 수 있음)

이미지 링크 : img 태그를 a 태그 안에 사용하면 이미지를 클릭했을 때 특정 링크로 이동하게 할 수 있음
