글로벌 속성 : 모든 태그에서 공통으로 사용할 수 있는 속성

전체 글로벌 속성 목록

자주 사용하는 글로벌 속성
속성 값 설명
class value 요소에 클래스 값 지정
클래스 값은 CSS에서 클래스 선택자로 활용
같은 클래스명은 여러 요소가 중복해서 가질 수 있음
id value 요소에 아이디 값 지정
아이디 값은 CSS에서 아이디 선택자로 활용
중복될 수 없음
style style 요소에 인라인 스타일 지정
CSS 코드를 인라인으로 작성할 때 사용
title text 요소에 추가 정보 지정
마우스를 요소 위에 올리면 툴팁으로 추가 정보가 표시됨
lang language code 요소에 사용한 텍스트의 언어 정보 지정
전체 언어 코드
hidden hidden 요소를 화면에서 감춤
data-* value 사용자가 임의의 속성을 만들 수 있음
(ex. data-name 태그, data-hero 태그 등)
