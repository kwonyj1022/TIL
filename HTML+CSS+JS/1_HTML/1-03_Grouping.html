그룹 짓기 : 공간 분할 태그 (div 와 span)

div 태그 : 블록 요소와 인라인 요소를 그룹으로 묶을 때 사용

div 태그는 텍스트를 영역별로 그룹을 지을 때 사용

HTML의 모든 태그는 class 속성을 사용할 수 있음
class 속성은 CSS를 적용하기 위한 식별자로 사용하는 속성

span 태그 : 인라인 요소를 그룹으로 묶을 때 사용

하나의 태그 안에 작성된 텍스트 일부분만 디자인을 다르게 적용하려면 태그 안에서 공간을 분할해야 함
이때 span 태그를 사용

CSS로 스타일을 적용하면 span 태그를 사용한 텍스트 부분에 디자인 변화를 줄 수 있음
